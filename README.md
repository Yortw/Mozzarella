# Mozzarella
Because it's stringy, like the cheese.

## What is it ?
A library for dealing with strings in .Net. Contains extension methods, and useful string related functions.

## Supported Platforms
Currently;

* .Net Framework 4.0
* Silverlight 5.0
* NetStandard 1.1
    * .Net Framework 4.5+
    * Windows Phone 8.0+ (NOT WP SL though) 
    * Xamarin.iOS 
    * Xamarin.Android
    * WinRT
    * UWP 10+
